# Landing Zone Copy (Standard Mode)

Dashboard-editable copy used when a campaign's zone schema sets `generation_mode` to `standard`. `generate.landing-zones` reads the section below matching each zone name and writes it, subject to the same length and fallback rules as personalized output, to that zone's property. Edit this guideline to change standardized copy without a deploy. Sections below match the example zones in `landing-zone-schema`.
